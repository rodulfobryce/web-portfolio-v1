Classic Models

This project showcases a full data pipeline using the popular Classic Models dataset. Starting with raw .csv data, I used Excel to clean, reformat, and segment the dataset into logical tables (Customers, Orders, Products). I then loaded the structured data into MySQL, where I built views and used CTEs to prepare it for reporting. Using an ODBC connection, I integrated the data into Power BI to design an interactive Sales Dashboard. The dashboard highlights key metrics such as: $9.6M Total Sales 326 Unique Orders $29.46K Average Order Value It includes visualizations of sales by product line, country, and region, with time-series charts and dynamic filters for deeper analysis. This project reflects my hands-on experience with the ETL process, data modeling, and business intelligence reporting.

PowerBi Excel MySQL

Netflix Content Dashboard

I built this dashboard to explore trends in Netflix’s global library using a dataset from Kaggle. It highlights how content grew rapidly around 2018–2019 before slowing down, with most shows rated for mature audiences (TV-MA, R). International movies and dramas lead in genre popularity, and Netflix shows strong global coverage across regions. The design follows Netflix’s visual branding. Covid Analysis in the Philippines

"This dynamic Power BI dashboard offers comprehensive insights into our country's COVID-19 data. Sourced from Kaggle, the dataset was cleaned in Excel before Power BI import. I leveraged DAX to create robust measures and custom age groupings, enhancing analytical depth. Intuitively structured, it features an 'Overview' page, complemented by three detailed sub-pages: 'Epidemiological Trends & Dynamics,' 'Demographics & Outcomes Analysis,' and 'Geographic Distribution & Hotspots.' This dashboard transforms raw data into actionable intelligence, showcasing my end-to-end capabilities in data processing, DAX modeling, and impactful visualization."
