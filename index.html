Certifications

UnionBank has partnered with my school and provided us with The Data Science Development Program certification, which provides in-depth training in Data Science, covering foundational and advanced topics. The course introduces key concepts in data analysis, machine learning, and artificial intelligence (AI), while also emphasizing ethical considerations and the business impact of data science. It includes modules on data preparation, exploratory data analysis, relational databases, programming languages (Python, SQL, R), and web scraping.

The FreeCodeCamp Responsive Web Design Certification teaches HTML for content and CSS for design. It covers basics like creating a cat photo app, advanced topics like CSS variables and accessibility, and responsive design techniques using Flexbox and CSS Grid for adaptable webpages.

This FreeCodeCamp JavaScript Certification covers core concepts like variables, arrays, functions, the DOM, and advanced topics such as OOP, Functional Programming, algorithmic thinking, local storage, and API data fetching. It teaches developers to build interactive web pages and manage data effectively. I am currently progressing onto this certification.
